a. ¿Quién inventó el sistema de control de versión Git y por qué?
b. ¿A quién pertenece actualmente Github y por qué?
c. ¿Hay otra forma que no sea la terminal para trabajar con Github?


   Solución

a. Linus Torvalds, porque se necesitaba conservar la integridad del código fuente gestionado. 

b. Microsoft GitHub Inc, porque lo adquirió con el mantra de "desarrolladores, desarrolladores, desarrolladores", por 7500 millones de dólares, así con el control de GitHub, Microsoft se hace también con la comunidad de desarrollo más nutrida del planeta.

c.  Se puede instalar la aplicación de GitHub para el escritorio. una vez instalada se puede seguir los paso a continuación:
   1. Se crea un repositorio desde el website de GitHub.
   2. Se abre la aplicación en el escritorio.
   3. Click en el icon mas ubicado en la esquina superior izquierda.
   4. De acuerdo a a lo que se requiera hacer dar click en Add, Create o Clone. 
   5. Seguir la secuencia de pasos que normalmente se seguiria desde la terminal pero de manera gráfica en la aplicación.
