
¿Quién inventó el sistema de control de versión Git y por qué?
	Github fue desarrollado por Chris Wanstrath, P. J. 
	Hyett, Tom Preston-Werner y Scott Chacon usando Ruby on Rails
	Este sistema se invento por la necesidad de trabajo colaborativo, 
	asincronico donde prima el control de versiones y modificaciones de un proyecto.

¿A quién pertenece actualmente Github y por qué?
	Microsoft, compró y se apodero de GitHub por la cantidad de 7500 millones de dólares, 
	ya que vio el potencial en la nueva era y en los desarrolladores, que estan escirbiendo el codigo del mundo.
