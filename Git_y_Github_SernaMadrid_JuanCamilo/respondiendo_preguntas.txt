a. ¿Quién inventó el sistema de control de versión Git y por qué?

  Este sistema fue creador por Linus Torvalds pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente
