a. ¿Quién inventó el sistema de control de versión Git y por qué?

El sistema de control de versión Git fue inventado por Linus Torvalds, porque buscaban permitir la ramificación, el etiquetado y
almacenar el historial de cambios durante el desarrollo web que realicen los programadores. (Fuente: Atlassian Bitbucket (2022). 
Aprende Git. Lugar de publicación: Atlassian Bitbucket. https://www.atlassian.com/es/git/tutorials/what-is-git). 

b. ¿A quién pertenece actualmente Github y por qué?

Github pertenece a Microsoft, esta empresa quiso unir fuerzas con Github para seguir priorizando a los desarrolladores y su compromiso
por la libertad, la apertura y la innovación. (Fuente: El Pais (2018). Microsoft compra GitHub, la mayor plataforma de software libre. 
Lugar de publicacion: El Pais. https://elpais.com/tecnologia/2018/06/04/actualidad/1528122972_585607.html).

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

Se podria trabajar desde el repositorio remoto, para no trabajar desde la terminal de Github local.
