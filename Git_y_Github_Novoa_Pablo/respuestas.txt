a. ¿Quién inventó el sistema de control de versión Git y por qué?

Linus Torvalds tenía un gestor de paquetes que pasó a ser de pago, por lo que el desarrollador principal de Linux decidió crear un nuevo gestor de control de versiones para su sistema, y que diera soporte a su core. De esta forma nació Git.
