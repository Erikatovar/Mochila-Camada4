a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un proyecto de código abierto, con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005.

¿Por qué? Git se ha diseñado con la principal prioridad de conservar la integridad del código fuente gestionado.
