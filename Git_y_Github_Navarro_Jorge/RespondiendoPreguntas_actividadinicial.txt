A. ¿Quién inventó el sistema de control de versión de Git y por qué?
R/ Según la investigación realizada, este software de control de versiones fue inventado por Linus Torvalds. Al momento de ser creado
Se pensó en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos
de código fuente. El propósito es llevar registro de los cambios en archivos de computadora, a la vez que posibilita la coordinación del trabajo
entre varias personas sobre archivos compartidos en un repositorio.
